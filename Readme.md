# Instructions

## MCP Client
To start client use command

`python MCPClient.py`

Client will automatically start MCP Server and fetch responses from server. So there is no need to start MCP server.

## MCP Server with MCP Inspector

to start MCP Server which can be tested with MCP Inspector run command

`npx @modelcontextprotocol/inspector python MCPServer.py`

This will start server and open MCP inspector in browser.
